windows下使用注意

windows下需要安装Oracle客户端Instant Client，要选择适配Oracle版本的instantclient

1) 下载链接：instantclient
2) 解压到一个目录，并添加到环境变量



3) 将instantclient文件夹下的oci.dll、oraocci11.dll、oraociei11.dll拷贝到Python主目录\Lib\site-packages
